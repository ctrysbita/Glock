# Glock

Glock is a clock powered by OpenGL.

## Keyboard & Mouse Events

    - ESC = Quit.
    - W = Go ahead.
    - S = Go back.
    - A = Go left.
    - D = Go right.
    - SPACE = Go up.
    - LEFT SHIFT = Go down.
    - H = Toggle shadow.
    - P = Toggle particles.

    - Mouse movement = Move perspective.
    - Mouse scroll up/down = Zoom in/out.
